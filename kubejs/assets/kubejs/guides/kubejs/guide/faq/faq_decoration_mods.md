# **Q: Will more decoration mods be added?**
**Q: Will more decoration mods be added?**

Short answer: no, there are already enough decoration options in the pack.

* No additional decoration mods are planned.
* The pack already includes a large number of decorative blocks and variants.
* Many of these blocks are hidden or harder to discover; use JEI to search thoroughly for building blocks.
* Before asking for more decoration mods, try searching JEI by tag, color, or material to see what you already have.
