# **Q: Lashmak had more quests. Where did they go?**

**Q: Lashmak had more quests. Where did they go?**

Short answer: They were removed because of a copyright issue.

* Some old quests had content that triggered a copyright strike, so they had to be removed.
* New, original quests are being written to cover the same progression.
* For now, use the remaining quests plus JEI/guideme to navigate that part of the pack.
